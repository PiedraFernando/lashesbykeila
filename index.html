Mis trabajos

Cejas

La realización de este sistema pretende resolver las necesidades básicas de un restaurante de pizzas que son: la administración de almacén y la comunicación de caja y cocina en tiempo real.

Fue realizado en solitario como parte de proyecto final de tesina, en un tiempo limite de 6 meses, Usando tecnologías como: Vue3, Mongodb y Nodejs, para su diseño se utilizo AdobeXD donde se plantearon todas las pantallas funcionales del sistema.

Para la planeación del proyecto se utilizó la metodología incremental, para que el usuario pudiera dar retroalimentación del sistema a medida que se entregaba cada incremento. A su vez se documentó toda la funcionalidad utilizando diagramas de actividad y especificando por medio de tablas los requisitos funcionales.

Pestañas

La creación de un sistema integral que une, la página web para darse a mostrar a los posibles clientes y un punto de venta que actualiza las existencias y permite estar al tanto del almacén, es lo que realiza esta aplicación web.

Esta siendo realizada en solitario para mi familia tratando de expandir y mejorar el negocio brindando una herramienta sencilla y practica para que todos la puedan utilizar.
